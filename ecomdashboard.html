E-commerce Dashboard

A responsive admin dashboard designed for managing products, orders, and customers efficiently. Built with React and Tailwind CSS, it offers a clean and intuitive interface for e-commerce management.

Features

  • Product management with CRUD operations
  • Order tracking and status updates
  • Customer management and analytics
  • Responsive and accessible UI

Technologies Used

React, Tailwind CSS, REST APIs, Node.js
